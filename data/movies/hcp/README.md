# HCP 7T movie stimuli — place files here

The Human Connectome Project 7T movie-watching stimuli (`MOVIE1`–`MOVIE4` `.mp4`s) are not
openly downloadable — obtain them from **ConnectomeDB** (free account + HCP Open Access Data
Use Terms) and drop the `.mp4` files into this folder, then run the refresh.

Full instructions + licensing: [`docs/EXTERNAL_STIMULI.md`](../../../docs/EXTERNAL_STIMULI.md).
